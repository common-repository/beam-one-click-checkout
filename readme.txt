== Installation ==
1. Go to Beam Checkout setting under WooCommerce > Settings > Payments > Beam Checkout.
2. Check "Enable Beam Checkout Gateway".
3. Contact Beam to obtain a necessary data (merchant ID, API key, Secret key).
4. (Optional) If the obtained key is a test key, "Test mode" must be checked.
5. (Optional) Check "Ask for customer details" to require user information (name, phone, address, etc.) at the checkout.
6. (Optional) Check "Custom placement" to enable shortcode function which allow you to pass a class name or place it wherever you want.
